# 廣島之旅內容維護說明

本資料夾是 hiroshima.html 的資料來源。

## 新架構

1. 結構化互動資料：trip-data.json
2. 基本文字補充：overview.md
3. 其他未結構化備註：other-info.md

## 更新方式

1. 若要調整航班、住宿、時程、交通、餐飲、伴手禮、checklist：修改 trip-data.json
2. 若要調整說明文字：修改 overview.md 或 other-info.md
3. 重新整理 hiroshima.html 即可看到最新內容

## 注意事項

- 請使用 UTF-8 編碼
- mapsUrl 建議使用 Google Maps 搜尋連結
- VJW 產生器需要住宿與電話等欄位，若空白會在頁面中提醒
